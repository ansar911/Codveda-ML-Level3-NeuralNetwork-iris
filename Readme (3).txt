# 🧠 Neural Network Classification - Iris Dataset (Level 3 - Codveda Internship)

## 📌 Project Overview

This project implements a feed-forward neural network using TensorFlow/Keras to classify flower species from the Iris dataset.

---

## 🧠 Objective

* Design a neural network architecture
* Train the model using backpropagation
* Evaluate performance and visualize training

---

## 📊 Dataset

* Iris Dataset
* Features:

  * Sepal Length
  * Sepal Width
  * Petal Length
  * Petal Width
* Target:

  * Species (setosa, versicolor, virginica)

---

## 🤖 Model Architecture

* Input Layer
* Hidden Layer 1 (ReLU)
* Hidden Layer 2 (ReLU)
* Output Layer (Softmax)

---

## ⚙️ Training

* Optimizer: Adam
* Loss Function: Sparse Categorical Crossentropy
* Epochs: 50

---

## 📈 Results

* Achieved very high accuracy (~95–100%)
* Model performed consistently across training and validation

---

## 📊 Visualization

* Plotted training and validation accuracy
* Plotted training and validation loss
